Sobre o Projeto

Esta API foi desenvolvida como parte do meu aprendizado em Node.js e Express. Ela inclui documentação interativa gerada automaticamente com Swagger UI.

Endpoints Disponíveis:

GET / - Retorna mensagem de boas-vindas
GET /user/{nome} - Retorna saudação personalizada

Como Usar

Para testar os endpoints, acesse a documentação interativa ou use ferramentas como Postman ou Insomnia.
